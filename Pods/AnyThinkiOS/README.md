# AnyThinkSDK_ios
在SDK版本在5.4.4以上版本，支持cocoapod方式引入AnyThinkSDK。<br>
在podfile文件中添加:<br>

	#核心库,必须添加
	pod 'AnyThinkiOS'
	#各平台的Adapter
	#pod 'AnyThinkiOS/AnyThink(平台名)Adapter'
	#例如 Facebook 按照您需要支持的平台，按需引入即可
	pod 'AnyThinkiOS/AnyThinkFacebookAdapter'
